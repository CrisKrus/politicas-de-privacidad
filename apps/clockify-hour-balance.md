# Política de privacidad de Clockify Hours Tracker

Clockify Hours Tracker es una extensión de Chrome que muestra el progreso de horas mensual y anual del usuario dentro del Dashboard de Clockify.

## Datos tratados

La extensión puede tratar los siguientes datos:

- API key de Clockify introducida por el usuario.
-  Configuración local de jornada: horas por día, días laborables, fecha de inicio, festivos, vacaciones, permisos y ajustes manuales.
- Entradas de tiempo obtenidas desde la API de Clockify para calcular las horas trabajadas.

## Finalidad del tratamiento

Los datos se usan exclusivamente para calcular y mostrar el progreso de horas del usuario dentro de Clockify.

## Almacenamiento

La configuración se guarda localmente en el navegador del usuario mediante chrome.storage. La extensión no mantiene una base de datos propia ni envía estos datos a servidores propios.

## Comunicación con terceros

La extensión se comunica con la API oficial de Clockify para obtener las entradas de tiempo del usuario. No vende datos, no comparte datos con fines publicitarios y no transfiere datos a terceros para fines ajenos a la funcionalidad principal.

## Seguridad

Las comunicaciones con Clockify se realizan mediante HTTPS. La API key de Clockify se almacena localmente en el navegador del usuario.

## Control del usuario

El usuario puede modificar o eliminar la configuración desde la propia extensión o desinstalar la extensión para eliminar los datos locales asociados.
